# **Blood Donation Website**

## **Project Overview**

The **Blood Donation Website** is a full-stack web application designed to connect blood donors with people in need. It allows users to **register as donors, search for donors by blood group and location**, and manage donation requests. The platform aims to simplify the blood donation process and encourage community participation.


## **Features**

* **User Registration & Login:** Donors and recipients can create accounts and manage profiles.
* **Donor Directory:** Search donors based on **blood group, city, or availability**.
* **Donation Requests:** Recipients can post requests and notify nearby donors.
* **Admin Panel:** Manage users, blood requests, and monitor activity.
* **Responsive Design:** Accessible on desktop, tablet, and mobile devices.


## **Technology Stack**

* **Frontend:** React.js, HTML, CSS, Bootstrap/Tailwind
* **Backend:** ASP.NET Core MVC & Web API
* **Database:** SQL Server
  

## **Usage**

1. Donors can **register**, fill in blood group, location, and contact info.
2. Recipients can **search donors** by blood group and city.
3. Admin can manage users, approve donations, and monitor requests.
4. All users receive notifications for relevant blood donation requests.


## **Future Improvements**

* Add **real-time notifications** for urgent requests.
* Implement **map integration** to locate nearest donors.
* Add **donation history and tracking** for donors.
* Enhance **analytics** for admins (e.g., blood group availability, active donors).
